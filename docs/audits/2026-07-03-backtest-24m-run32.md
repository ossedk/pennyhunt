# Backtest run #32 — 24-month flagship (third regime) — 2026-07-03

## Setup

Same configuration as run #31 (threshold 0.65, ≥3 daily mentions, 30d
baselines, 3d cooldown, volume z ≥ 2 gate, entry ≤ $5, 10% stop / no take,
5% friction), extended to the full 24-month archive:

- **Window: 2024-08-02 → 2026-06-24** — deliberately starts 30 days after
  the archive begins (2024-07-03) so rolling baselines carry real history
  from the first candidate day. No warm-up artifact this time.
- 5,807 tickers evaluated, 4,672 with bars, 32,619 events, **719 fired
  signals** (vs 292 in the 12-month run).
- Data prep: 25-month bar sync for 4,708 tickers; 14 reverse-split zombies
  (ADTX, AREB, BINI, CDT, CETX, HUBC, ONCO, RUBI, SMX, SONG, TRNR, VCIG,
  VMAR, WHLR) initially failed on numeric overflow from absurd Yahoo
  split-adjusted prices (10^17) — fixed in `YahooMarketData` (bars with
  prices > 10^9 dropped as untradeable) and all 14 resynced.

## Headline results

| Metric | Run #32 (24m) | Run #31 (12m) |
|---|---|---|
| Signals | 719 | 292 |
| Hit rate (≥+30%/5d) | **13.1%** | 12.7% |
| Base rate | 3.9% | 2.9% |
| Precision lift | **3.4×** | 4.4× |
| Avg exit return (gross) | +0.49% | +2.1% |
| Net per trade (after 5% friction) | **−4.51%** | −2.88% |
| Profit factor | 0.59 | 0.74 |
| Stop rate | 64.0% | 61% |
| Prediction share (fired before move) | 46.5% | ~48% |

## Confidence (walk-forward, 15 features, monthly refits)

Quintile calibration on 31,768 scored events — **monotone again**:

| Quintile | Predicted | Realized |
|---|---|---|
| q1 | 2.8% | 1.6% |
| q2 | 3.9% | 2.7% |
| q3 | 4.8% | 3.7% |
| q4 | 6.2% | 4.3% |
| q5 | 10.4% | **8.3%** |

q5/q1 realized = **5.3×** (was 4.4× on 12 months). Brier 0.0395 vs reference
0.0394 — the model ranks reliably; its absolute level adds nothing over the
base rate, consistent with every prior run.

## Portfolio simulation (709 tradable signals)

- **Equal weight: catastrophic.** −80.5% total return, 86.7% max drawdown.
  Trading every signal at 5% equity for 24 months is wealth destruction.
- **Kelly (full and half): refused the strategy.** Took 22 of 709 trades
  (97% skipped), finishing −3.6% with 6.5% max drawdown. As in run #31, the
  as-of Kelly math correctly recognizes negative expectancy and stays in
  cash.

## Verdict

The third regime **confirms and hardens the run #31 conclusion**:

1. **Ranking skill is real and stable across all three regimes** — 3.4×
   precision lift, 5.3× monotone confidence separation. The system finds
   winners at far above chance.
2. **The configuration is not tradable** — net −4.5%/trade at 5% friction,
   PF 0.59, 64% of trades stopping out. 2024 H2 (new regime) did not rescue
   it; more data made the negative expectancy estimate tighter, not better.
3. The blocker remains **signal quality and selectivity**, not evidence.
   719 signals is enough to train the non-linear model (Phase C); the
   LLM post-type features and author track records are the next inputs.
