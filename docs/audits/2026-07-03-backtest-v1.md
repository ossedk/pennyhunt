# Backtest v1 — Validation Notes (2026-07-03)

> First full-history replay of the production signal engine over ~6 months of
> archived forum data. This is the initial Phase 4 data point, not the final
> decision-gate verdict.

## Data

- **Posts**: 160,339 (146,334 backfilled Jan 3 → Jul 1 via Arctic Shift, rest live via Apify), all processed through the production pipeline (TickerExtractor + LexiconSentiment).
- **Mentions**: 85,357 across 4,205 distinct tickers (2,825 with usable Yahoo daily bars; the rest are delisted/OTC-unknown and excluded from grading on both signal and control side).
- **Market data**: 404,504 daily OHLC bars (Yahoo, keyless).

## Methodology (guardrails against self-deception)

- As-of rolling 30-day baselines per ticker — z-scores computed only from data before each day. No look-ahead.
- Same scoring formulas as the live engine (`SignalMath`, shared code). Cross-source component unavailable pre-July → weights renormalized over acceleration/breadth/sentiment.
- Entry at the **next trading day's open** after the signal day.
- Hit = any close within 5 sessions ≥ +30% above entry.
- Control group = all mention-days (≥ min mentions) that did **not** fire.
- Reaction vs prediction: signals where price already ran >15% in the 3 sessions before entry are labelled "reaction".

## Results

| | v1 params (thr 0.65, ≥3 mentions) | stricter (thr 0.75) |
|---|---|---|
| Simulated signals (6 mo) | 3,277 | 1,632 |
| Hit rate (≥+30% in 5d) | **5.1%** | 5.0% |
| Control base rate | 3.2% | 4.2% |
| Avg 5d return | **+3.6%** | +5.7% |
| Control avg 5d | −1.4% | +4.1% |
| Median 5d return | −0.7% | −0.9% |
| p10 / p90 5d | −17.6% / +14.0% | −19.6% / +12.4% |
| Positive-5d share | 45.1% | 45.1% |
| Prediction share | 85% | 82% |
| Hit rate, predictions only | 3.5% | 3.1% |
| Hit rate, reactions only | 14.1% | 13.7% |

## Honest reading

1. **There is a real but modest edge in the averages.** v1 signals average +3.6%/5d vs −1.4% for non-fired mention-days — a ~5pp spread on 3,277 events. The stricter threshold's control group also improves, so the spread there is thinner (+1.5pp).
2. **The edge is fat-tail-driven, not consistent.** Median signal loses −0.7%; the mean is carried by a small number of huge winners (top outcomes +58% to +934%). A strategy trading this needs to survive many small losses to catch few large wins — position sizing and slippage decide whether it's tradable at all.
3. **The "prediction" bucket underperforms the "reaction" bucket on hit rate** (3.5% vs 14.1%). Buzz that fires *after* a move has started is more likely to precede a further ≥+30% pop than buzz alone. That is momentum-confirmation, not pure prediction — an important negative finding for the core hypothesis as currently scored.
4. **Not yet accounted for:** spreads (penny stocks routinely 2–10%), liquidity limits, and the fact that next-day-open entry on gappers already gives up part of the move. A PnL simulation with haircuts is the required next step before any decision-gate call.
5. **Sample caveats:** one 6-month window (Jan–Jun 2026), one market regime; lexicon-only sentiment (FinBERT/LLM tiers not yet live); score=0 for backfilled posts is not used by scoring but breadth/sentiment quality varies with archive completeness.

## Next steps (Phase 4 continuation)

1. PnL simulation with spread/liquidity haircuts and capped position sizes.
2. Walk-forward weight fitting (logistic regression on the 3,277 graded events; current weights are hand-picked).
3. Test market-confirmation gating (only fire when volume z-score confirms but price hasn't run yet) — aimed directly at improving the prediction bucket.
4. Re-run on a second window (2025 H2 backfill) to check regime robustness.
